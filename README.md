# Learn OpenGl

> 运行环境：mac c++

> 参考资料：https://learnopengl-cn.github.io

test_triangle: 绘制三角形

test_rectangle: 绘制矩形

test_rotate: 绘制矩形纹理，随时间改变自动旋转

test_overlying: 两个图形叠加

test_rgb: 渲染RGB图

test_yuv: 渲染YUV420P图

test_sphere：绘制球体网格框架，采用线框模式

test_fullview: 全景图

test_fullview2: 在给定位置观察全景图，球体内含一张图片

test_skybox: 天空盒

test_compute: 采用opengl做通用计算（compute shader）
